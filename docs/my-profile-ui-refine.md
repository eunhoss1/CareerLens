# 내 프로필 UI 개선 정리

## 작업 목적
내 프로필 화면을 더 보기 쉽고 정돈된 프로필 관리 화면으로 개선한다.

## 수정 범위
- `frontend/app/onboarding/profile/page.tsx`
- 프론트엔드 프로필 화면 UI만 수정
- 백엔드, API, DB, 인증 로직, 라우팅 구조는 수정하지 않음
- 공통 입력 컴포넌트 변경은 제거하고, 프로필 화면 내부 전용 입력 스타일로 제한함

## 주요 변경 내용
- 상단의 불필요한 상태 카드 제거
- 상단 영역을 `내 프로필` 제목, 설명, 저장 버튼 중심으로 간소화
- 좌측 보조 영역을 작성 현황과 추천 기준 설정만 남기도록 정리
- 입력 섹션 카드의 여백, 제목, 설명, border 톤 정리
- 프로필 화면 전용 입력/셀렉트 UI 추가
- 추천 항목은 접힘 UI로 유지하되 사용자가 찾기 쉽도록 버튼형 패널로 개선
- 지원 조건과 증빙 섹션을 `지원 조건`과 `증빙 자료` 두 그룹으로 분리
- 모바일과 데스크톱에서 카드가 자연스럽게 쌓이도록 반응형 grid 유지

## 유지한 것
- 기존 프로필 저장 로직
- 기존 API 호출 구조
- 기존 저장 후 이동 경로
- 기존 인증 흐름
- 기존 라우팅 구조
- 백엔드 코드 미수정

## 인터랙션 정리
- 저장 버튼 hover 효과 유지
- 추천 기준 토글 선택 상태 표시 유지
- 추천 항목은 접힘/펼침 방식으로 노출
- 입력/셀렉트 focus 시 border와 ring으로 현재 입력 위치 표시

## 확인 결과
- `npm run typecheck` 통과
- `git diff --check` 통과
- `npm run build` 통과

## 확인 필요 사항
- 내 프로필 화면에서 저장 버튼 이동 확인
- 지원 조건/증빙 섹션 배치 확인
- 추천 항목 접힘 UI 확인
- 모바일 화면에서 섹션이 깨지지 않는지 확인
