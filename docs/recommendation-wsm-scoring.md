# CareerLens Weighted Sum Method 적용 정리

## 적용 판단

Weighted Sum Method(WSM)는 여러 평가 기준을 같은 척도로 정규화한 뒤, 각 기준의 중요도 가중치를 곱해 대안을 비교하는 다기준 의사결정 방식입니다.

CareerLens의 추천 진단은 공고 후보를 여러 기준으로 비교합니다. 현재 각 기준 점수는 0~100 범위로 정규화되어 있으므로 WSM을 적용하기에 적합합니다.

## CareerLens 평가 항목

추천 공고마다 아래 항목을 0~100 점수로 정규화합니다.

- 합격 가능성
- 연봉 매력도
- 워라밸
- 기업 가치
- 직무 적합도

이후 공고별 기본 가중치와 사용자가 마이페이지에서 선택한 우선순위를 합쳐 최종 가중치를 만듭니다.

## 계산 방식

```text
FinalScore = sum(normalizedScore_i * normalizedWeight_i)
```

예시:

```text
합격 가능성 80점 * 0.30
연봉 매력도 60점 * 0.15
워라밸 70점 * 0.15
기업 가치 90점 * 0.15
직무 적합도 75점 * 0.25
= 76점
```

## 구현 위치

- `WeightedSumScoringService`
  - WSM 공식 기반 최종 점수 계산
  - 점수는 0~100 범위로 보정
  - 가중치 합이 100이 아니어도 내부에서 비율로 정규화

- `RecommendationServiceV2`
  - 공고, PatternProfile, 사용자 프로필 비교 결과를 기준별 점수로 만든 뒤 WSM 서비스에 전달
  - 기존 API 응답 구조와 DB 저장 구조는 유지

## 발표 설명 문장

CareerLens는 단순 키워드 매칭이 아니라 합격 가능성, 연봉, 워라밸, 기업 가치, 직무 적합도처럼 서로 다른 평가 기준을 0~100 점수로 정규화한 뒤 Weighted Sum Method로 종합 점수를 계산합니다.

사용자가 중요하게 보는 항목을 선택하면 해당 기준의 가중치가 올라가고, 최종 추천 순위가 사용자 우선순위에 맞게 달라집니다.

## 한계와 확장 방향

WSM은 설명 가능하고 구현이 단순하다는 장점이 있지만, 기준 간 비선형 관계나 문장 의미 유사도를 직접 반영하지는 못합니다.

향후에는 공고 본문, 사용자 프로젝트, 이력서 문장을 임베딩으로 변환한 뒤 유사도 분석 결과를 `직무 적합도` 또는 `합격 가능성`의 하위 점수로 반영할 수 있습니다.
